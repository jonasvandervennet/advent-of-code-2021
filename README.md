# Advent Of Code 2021

A Rust implementation of the 2021 edition of [Advent of Code](https://adventofcode.com/2021/).

## Build

```sh
cargo build
```

## Test

```sh
cargo test
```

## Run

### Run all

```sh
cargo run
```

### Run a specific day

```sh
cargo run -- --day 1
```
